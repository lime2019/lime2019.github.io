技术专利

lime2019 2020-09-27 专利

本文为飞桨PaddlePaddle在哔哩哔哩投稿的飞桨开发者说live：技术专利申请不踩坑视频的总结。

学校大创结题需要申请软著、专利等，看到这个科普视频还是很有帮助的。

# 专利的价值和意义

专利的价值：

  • 能力体现
    • 技术、科研能力的一个体现
  • 成果转化
    • 专利转让、许可
    • 促成商业合作
  • 职业发展助力
    • 求职
    • 职称评审
  • 其他
    • 可申请落户政策（某些城市）
    • ...

对公司的价值：

  • 提升市场竞争力
    • 体现技术优势
    • 增强企业资质
  • 获取收益
    • 专利转让
    • 专利许可
  • 保护自身业务的自由发展
    • 保护创新成果
    • 遏制侵权行为
  • 无形资产
    • 质押融资
    • 专利技术出资

# 专利是什么

《专利法》第十一条
发明和实用新型专利权被授予后，除本法另有规定的以外，任何单位或者个人未经专利权人许可，都不得实施其专利，即不得为生产经营目的制造使用、许诺销售、销售、进口其专利产品，或者使用其专利方法以及使用、许诺销售、销售、进口依照该专利方法直接获得的产品。
外观设计专利权被授予后，任何单位或者个人未经专利权人许可，都不得实施其专利，即不得为生产经营目的制造、许诺销售、销售、进口其外观设计专利产品。

专利的三个特性：

  • 排他性
    • 权利专属于权利人所有
    • 禁止他人未经许可而实施其专利
  • 地域性
    • 仅在特定的国家或地区的地域范围内有效
    • 不具有域外效力
  • 时间性
    • 仅在法定的保护期内有效

《专利法》 第二条
本法所称的发明创造是指发明、实用新型和外观设计。

专利有效期：

  • 发明专利
    • 保护产品、方法
    • 期限20年
  • 实用新型专利
    • 保护产品构造
    • 期限10年
  • 外观设计专利
    • 保护产品的形状、图案、色彩及其结合
    • 期限10年

开源与专利：

不同的开源许可证的专利条款不同；

Apache2.0许可证中有明确的专利许可条款：

  • 在此许可证下，专利不影响开源开发者对代码的使用
  • 同时，专利可保护开源社区开发者的创新成果，保护开源生态系统

# 如何申请专利

专利申请流程：

  1. 技术方案
    • 发明人提出技术方案，并由自己或委托代理机构撰写申请文件
  2. 申请
    • 向国家知识产权局提交申请文件及缴纳相关费用
  3. 审查
    • 初步审查：手续、文件、格式等
    • 实质审查（发明）：新颖性、创造性、实用性等
  4. 授权/驳回
    • 授权：由国家知识产权局向个人/企业颁发证书
    • 驳回：发出驳回通知书

专利申请要求：

  • 形式要求
    • 文书
      • 请求书
      • 五书
        • 权利要求书：对专利权权力范围的声明
        • 说明书：对技术方案的详细说明
          • 技术领域
          • 背景技术
          • 发明内容
          • 附图说明
          • 具体实施方式
        • 说明书附图：对说明书的图示说明
        • 说明书摘要：专利公开文本首页中显示
        • 摘要附图
    • 格式
      • 各部分要求
    • 流程
      • 缴纳费用
      • 期限
  • 实质要求
    • 客体条件
      • 是否属于专利保护客体
    • “三性”
      • 新颖性
        • 没有在国内或国外出版物上公开发表
        • 没有在国内或国外公开使用或其他方式为公众所知
      • 创造性
        • 现有技术（申请日之前）
        • 突出的实质性特点（非显而易见）
        • 显著的进步（有益的技术效果）
      • 实用性
        • 能制造
        • 能使用
        • 能产生积极效果
        • 不违反自然科学规律（永动机）
    • 其他法条

专利代理的概念：

专利申请人（或者专利权人）委派具有专利代理师资格的在专利局正式授权的专利代理机构中工作的人员，在委托权限内，以委托人的名义，按照《专利法》、《专利代理条例》的规定，为委托人办理专利申请或其它专利事务。

# 常见问题

自主研发的成果不申请专利就有知识产权吗？

两个以上的申请人分别就同样的发明创造申请专利的，专利权授予最先申请的人。——专利法第九条第二款

现在只是构想，没有实际产品，因此没有必要也不能申请专利吗？

  • 专利要求技术方案“能够实现”，并不要求有实际产品
  • 前瞻布局

使用飞桨开发的可以申请专利吗？

只要有创新的idea，就可以申请专利（符合专利的形式和实质要求）

向国外申请专利难吗？有什么特殊流程？

  • 申请途径：PCT（《专利合作条约》）途径（国际申请）& 巴黎公约途径
  • 申请要求：必须要经过国内的保密审查
  • 费用：国外官费+涉外代理费（律师费）

# Q&A

算法可以申请专利吗？

纯粹的算法是不能受到中国专利法的保护，如果通过这个算法解决实际的技术问题，获得了技术效果，并采用了技术手段，是可以受到技术保护的。

申请专利可以与论文投稿同时进行吗？

先申请专利再发表论文。

查询是否已存在专利？

通过国家知识产权局官网进行检索。

可不可以同时申请实用性和发明性专利？

可以，但是如果实用性专利先得到授权，后续得到发明性专利后需要在二者中选取一个。

纯软件是否可以申请实用性专利？

实用性专利只保护对产品结构上的改进，也就是硬件结构上的保护。
